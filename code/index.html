Ruby on Rails

In 2007 I discovered Basecamp which lead me to Ruby on Rails (Rails). Rails inspired me to build my own project management tool which has taken longer than anticipated :-) I test with Minitest, monitor with New Relic, and deploy with Heroku. I am excited for Action Cable which will ship as part of Rails 5.

JavaScript

I prefer vanilla JS over CoffeeScript. The Rails apps I've worked with tend to use JQuery, Underscore, BackboneJS (and/or) AngularJS. I use Jasmine for testing. I am excited about the future of BabelJS, ECMA 6, and ReactJS.

Middleman

I build static sites with Middleman because it is very similar to Rails. It's written in Ruby, has an Asset Pipeline, local server, and gems for developing with sophisticated tools. Then, with free static site hosting from GitHub that makes deployment easy and free!

CSS

I enjoy thoughtbot's front-end suite (Bourbon, Neat, and Bitters). I write SCSS and regularly try to apply methodologies like SMACSS and BEM as best I can.

HTML

HTML could be defined in broader terms as View management to include working with JS frameworks, Rails partials (ERB/SLIM), responsive web design, and cross browser testing.
